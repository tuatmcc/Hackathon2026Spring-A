# スキルツリー画面

複数のスキルツリーを表示できるようにする．
この画面に関しては，あとから修正が入る可能性があるので，できるだけ汎用的に作成する．

## 画面構成

- 小さいスキルツリーを複数並べる
  - スキルツリーのノードをクリックすると，そのスキルツリーのノードを選択できる．選択されたノードの背景色を変える．
- スキルポイントの欄

## スキルツリーの種類

- 隠れ層ツリー
  - 層の種類 (全結合(Dense) / 畳み込み(Conv2D) / Flatten など...)
  - 隠れ層数
  - 層の幅
- 活性化関数ツリー
  - 活性化関数 (Sigmoid / ReLU / GELU)
- 学習手法ツリー
  - 学習手法 (SGD / Adam)
- 正則化ツリー
  - 正則化 (L1 / L2 / Dropout)
