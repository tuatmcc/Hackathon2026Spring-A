# ゲーム画面

[A Neural Network Playground](https://playground.tensorflow.org/) を参考にする．

## 画面構成

- ステージ選択項目
- 学習条件設定
- NN 構成 (できればGUIでコネコネ操作できるようにする)
- 学習実行ボタン
- 学習結果表示

### ステージ選択項目

- ステージ名
- ステージ説明
- ステージ難易度
- ステージクリア条件

### 学習条件設定

- 学習手法 (スキルで解放)
- 学習率 (自由値)
- バッチサイズ (自由値)
- エポック数 (固定値)

### NN 構成

- 層の種類 (スキルで解放)
- 層の幅 (スキルで解放)
- 活性化関数 (スキルで解放)
- 正則化 (スキルで解放)

ただし，最終層の活性化関数やLoss関数はステージによって固定される．

### 学習実行ボタン

- 学習実行ボタン
  - 学習実行中の表示
  - 学習完了後の表示

### 学習実行完了後の表示

- train loss / val loss
- train accuracy / val accuracy
- 決定境界
- データ点

## クリア条件

各ステージで validation accuracy が所定の閾値以上になればクリア．

## ステージ候補例

- Linear
- XOR
- Spiral
- MNIST
